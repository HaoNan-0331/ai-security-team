# 客户端版本更新记录

## 版本历史

### v2.1.0 (2026-01-30)

#### 新增功能
- **Telnet 执行器** - 新增 Telnet 命令执行功能，支持连接到不支持 SSH 的旧设备
  - 支持 Telnet 连接认证（用户名/密码）
  - 可配置登录提示符、密码提示符和 Shell 提示符
  - 支持 telnetlib3 2.0.8 版本
  - 提供同步和异步两种执行方式

- **SSLAdapter** - 新增自定义 SSL 适配器，解决 HTTPS 连接兼容性问题
  - 使用 TLSv1.2 协议
  - 兼容自签名证书
  - 处理旧版本服务器的 SSL 握手问题

#### 功能改进
- **用户认证** - 实现完整的用户登录认证机制
  - 启动时需要输入用户名和密码
  - 获取访问令牌（24小时有效期）
  - 支持令牌自动刷新

- **配置管理** - 优化配置管理功能
  - 首次运行时提供配置向导
  - 支持交互式配置修改
  - 配置格式验证

#### 技术更新
- 更新依赖包版本：
  - telnetlib3: 2.0.8
  - websockets: 13.1
  - requests: 2.32.3
  - paramiko: 3.5.0

#### 文件变更
- 新增文件：
  - `telnet_executor.py` - Telnet 执行器模块
  - `CHANGELOG.md` - 版本更新记录

- 修改文件：
  - `main.py` - 添加 SSLAdapter 和用户认证
  - `websocket_client.py` - 添加 Telnet 任务处理
  - `requirements.txt` - 更新依赖版本

### v2.0.0 (2026-01-26)

#### 新增功能
- **分布式命令执行** - 基本功能实现
  - SSH 命令执行（支持华为、H3C、思科等设备）
  - HTTP 请求执行（GET/POST/PUT/DELETE）
  - 本地命令执行（Windows/Linux）
- **WebSocket 通信** - 与服务端保持长连接
- **自动重连** - 连接断开后自动重连
- **本地历史记录** - 保存最近 6 个月的执行记录
- **识别码认证** - 客户端唯一标识

#### 技术栈
- Python 3.12+
- WebSocket（异步通信）
- Paramiko（SSH）
- Requests（HTTP）
- SQLite（本地数据库）

## 版本说明

### 版本命名规则
- 主版本号 (Major): 重大架构变更
- 次版本号 (Minor): 新增功能
- 修订号 (Patch): Bug 修复

### 兼容性
- 向下兼容 API v2.0+
- 客户端需要与服务端版本匹配
