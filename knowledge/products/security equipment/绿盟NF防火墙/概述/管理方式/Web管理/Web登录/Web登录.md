# Web登录

本节以Firefox浏览器为例，介绍登录NF的Web管理系统的步骤。

确认管理主机与NF正常通讯（可使用Ping方法）。

NF默认禁止使用Ping，需在Web中开启访问控制。

打开浏览器，用HTTPS方式连接NF的管理IP地址，例如：https://192.168.1.1。

回车后显示安全报警信息，如图1-6所示。

登录前的安全报警

单击【高级】按钮，页面如图1-7所示。

查看高级选项

选择【接受风险并继续】，跳转到NF的Web登录页面，如图1-8所示。

Web登录

输入用户名和密码。

初次登录可使用缺省系统管理员登录，初始帐号和密码请参见 Web管理员初始帐号。

单击【登录】按钮，系统会强制要求修改初始密码。

密码修改完成后，导入证书，然后进入Web管理页面。

Web管理页面的具体操作方法请参见本文档后续章节。
