# Web管理

Web管理系统为管理员提供了更直观的人机交互方式，管理员通过Web管理界面实现对NF的管理和配置。下面详细介绍Web管理的用户、登录方法、Web页面布局以及常用操作。

在NF的Web管理界面下，通过三权分立模式进行管理，登录设备的Web用户的权限是由角色决定的。系统内置了3种角色：系统管理员、安全策略管理员和安全审计员，分别均有1个默认帐号weboper、webpolicy和webaudit。

根据NF是否支持虚拟系统功能，3种内置角色的权限说明分别见表1-1、表1-2和表1-3。在支持虚拟系统功能的情况下，根系统为NF中缺省存在的特殊的虚拟系统。

本节以Firefox浏览器为例，介绍登录NF的Web管理系统的步骤。

确认管理主机与NF正常通讯（可使用Ping方法）。

NF默认禁止使用Ping，需在Web中开启访问控制。

打开浏览器，用HTTPS方式连接NF的管理IP地址，例如：https://192.168.1.1。

回车后显示安全报警信息，如图1-6所示。

登录前的安全报警

单击【高级】按钮，页面如图1-7所示。

查看高级选项

选择【接受风险并继续】，跳转到NF的Web登录页面，如图1-8所示。

Web登录

输入用户名和密码。

初次登录可使用缺省系统管理员登录，初始帐号和密码请参见 Web管理员初始帐号。

单击【登录】按钮，系统会强制要求修改初始密码。

密码修改完成后，导入证书，然后进入Web管理页面。

Web管理页面的具体操作方法请参见本文档后续章节。
