# URL过滤日志

通过URL过滤日志，可以查看用户访问URL产生的日志信息，以便于根据用户的访问情况调整URL过滤策略，规范用户的上网行为。

只有在URL过滤模板 中开启“记录日志”功能，并且在 防火墙策略 中引用相应的对象后，URL过滤日志的展示功能才会生效。当报文与策略成功匹配后，就会产生URL过滤日志。

进入 日志报表 > URL过滤日志 页面，默认展示最近1小时的URL过滤日志。可以设置查询条件进行日志查询。支持日志导出和自定义日志列表列信息的操作。URL过滤日志查询条件说明如表8-11所示，URL过滤日志展示内容如表8-12所示。
