# WEB应用通用防护日志

通过查看WEB应用通用防护日志，可以分析WEB应用通用防护事件，从而采取防护措施保障网络安全。

只有在 Web应用通用防护模板 中对特定的事件开启“记录日志”功能，并且在 防火墙策略 中引用相应的对象后，WEB应用通用防护日志的展示功能才会生效。当报文与策略成功匹配后，就会产生WEB应用通用防护日志。

进入 日志报表 > WEB应用通用防护日志 页面，默认展示最近1小时的Web应用通用防护日志。可以设置查询条件进行日志查询。支持日志导出和自定义日志列表列信息的操作。Web应用通用防护日志查询条件说明如表8-19所示，Web应用通用防护日志展示内容如表8-20所示。
