# Web应用通用防护模板

进入 策略 > 对象 > WEB应用通用防护 > WEB应用通用防护 页面，可以配置Web应用通用防护模板以及开启Web应用通用防护功能。

开启Web应用通用防护

X-Forwarded-For（简称XFF）是用来识别通过HTTP代理或负载均衡方式连接到Web服务器的客户端最原始的IP地址的HTTP请求头字段。

开启后，NF将封禁所有XFF请求。

配置Web应用通用防护模板

单击【新建】按钮，配置Web应用通用防护模板参数，即可新建Web应用通用防护模板。单击【应用配置】按钮，使配置生效。Web应用通用防护模板参数说明如表6-53所示。

新建Web应用通用防护模板后，可以进行查看规则详情、Web应用防护溯源、编辑和删除操作。

Web应用通用防护模板参数

Web应用防护溯源

在Web应用通用防护模板列表中，单击溯源列的即可查看对应模板的全部WEB应用防护记录，包括防护时间、规则ID、防护类型、源IP、目的IP、源端口、目的端口、威胁等级、溯源IP和防护动作。
