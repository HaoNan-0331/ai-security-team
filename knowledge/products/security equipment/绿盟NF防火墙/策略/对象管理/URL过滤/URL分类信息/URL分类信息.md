# URL分类信息

NF使用具有业界领先的中、英文URL分类库，内含按照不同类型（例如：赌博、暴力、恶意软件等）划分的超过1000万条记录的URL信息，可以实现对工作无关网站、不良信息、高风险网站的准确高效过滤。

对URL进行分类检测时，NF中的URL分类库为离线模式，从设备上获取URL分类信息。

进入 策略 > 对象 > URL过滤 > URL分类信息 页面，URL分类信息列表包括系统分类和自定义分类，如图6-14所示。

URL分类信息

查询URL系统分类

URL系统分类支持模糊查询，输入关键词，回车后即可查看匹配的URL分类信息。

自定义URL分类

在如图6-14所示的页面中，在“自定义分类”区域，单击【新建】按钮，配置自定义URL分类参数。单击【确定】按钮，保存URL分类信息。单击【应用配置】按钮，使URL分类生效。自定义URL分类参数说明如表6-45所示。
