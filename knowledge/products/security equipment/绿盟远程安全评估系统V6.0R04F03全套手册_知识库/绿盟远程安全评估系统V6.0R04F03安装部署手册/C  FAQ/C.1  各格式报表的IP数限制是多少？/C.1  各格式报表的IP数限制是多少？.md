C.1 各格式报表的 IP 数限制是多少？
A：HTML报表输出最大主机数为65535；Word和pdf输出最大主机数为12000。
C.2 如何在跨网段访问或扫描时配置缺省路由？
A：全零默认路由不需要手动编写，可通过设置网口的缺省网关来自动生成。扫描口默认路
由可通过web界面或串口进行配置，管理口网络及路由配置只能在串口下操作。
Web界面配置方法：
进入 系统管理 > 配置，在网络页签中选择需要生成默认路由的网口，单击【编辑】，在弹
出的网络配置框中，输入网口IP地址及网关，缺省网关选择“是”，配置完成后单击【确定】，
如图 C-1所示。
图 C-1 配置缺省路由
串口配置方法：
单击网络配置设置默认网关，在IPV4网关一栏设置需要配置默认路由的接口数字，例如：0
代表管理口，1代表扫描口1，2代表扫描口2，以此类推，如图 C-2所示。
