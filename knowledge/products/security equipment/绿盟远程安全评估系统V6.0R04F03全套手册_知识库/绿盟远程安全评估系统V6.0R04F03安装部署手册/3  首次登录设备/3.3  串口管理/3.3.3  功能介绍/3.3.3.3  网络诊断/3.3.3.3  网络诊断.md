图3-13 控制台管理 – 删除管理口路由
管理员删除RSAS的管理口路由时，需要注意以下几点：
⚫ 在删除的路由信息前，会显示当前接口下的路由表，选择需要删除的路由。
⚫ 配置完毕，选择“ok”，删除的路由即刻失效。
添加扫描口路由
在控制台管理的主菜单中，进入 网络配置 > 添加扫描口路由，添加扫描口路由的方法与添
加管理口路由相同，请参考 添加管理口路由 的介绍。
删除扫描口路由
在控制台管理的主菜单中，进入 网络配置 > 删除扫描口路由，删除扫描口路由的方法与删
除管理口路由相同，请参考 删除管理口路由 的介绍。

图3-14 控制台管理 – 网络诊断

表3-6 控制台网络诊断工具
工具 作用
Ping 用于检测RSAS的到达目标主机的连通性，支持IPv4和IPv6地址。
路由追踪 用于检测RSAS到达目标主机所经过路由的跳数。
网络状态 用于查看RSAS的网络连接状态。
显示路由信息 用于查看RSAS与路由设备接口的网络参数。
DNS查询 用于查询DNS域名解析信息。
