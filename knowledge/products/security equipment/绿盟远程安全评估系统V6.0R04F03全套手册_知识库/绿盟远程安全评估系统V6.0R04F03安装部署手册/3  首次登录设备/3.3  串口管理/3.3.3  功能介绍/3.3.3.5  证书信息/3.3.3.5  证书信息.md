3.3.3.5 证书信息
在控制台管理的主菜单中，进入 证书信息，如图3-16所示。
显示了当前系统安装的授权证书文件信息，包括产品类型、证书起始日期、终止日期（证书
失效时间）、证书选项（购买模块）等。
