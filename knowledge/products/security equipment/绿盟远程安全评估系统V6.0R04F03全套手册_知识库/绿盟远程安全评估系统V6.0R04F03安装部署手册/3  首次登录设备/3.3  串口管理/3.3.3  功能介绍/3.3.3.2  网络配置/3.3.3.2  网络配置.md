3.3.3.2 网络配置
RSAS包含扫描网络接口和管理网络接口。其中，扫描接口用于对网络进行扫描，管理接口
用于管理RSAS设备。管理员可以直接使用扫描接口管理RSAS设备及进行任务的评估。
配置 DNS
在控制台的主菜单中，进入网络配置 > 配置DNS，如图3-8所示。
图3-8 控制台管理 – 配置DNS
管理员配置RSAS的DNS时，需要注意以下几点：
⚫ 至少配置一个DNS。
⚫ DNS地址1为必填项。
⚫ 若需RSAS设备在线升级，DNS服务器必须正确配置。
设置默认网关
在控制台的主菜单中，进入网络配置 > 设置默认网关，如图3-9所示。

图3-9 控制台管理 – 设置默认网关
在网关一栏设置需要生成配置缺省路由的接口数字，0代表管理口，1代表扫描口1，2代表
扫描口2，以此类推。例如设置为1，将在扫描口1上生成一条0.0.0.0 0.0.0.0 eth1的缺省路
由。若不设置，系统将默认网口1对应的网关作为默认网关。
通过设置默认网关生成缺省路由的前置条件为对应网络接口配置中需正确填
写网关地址。具体配置可参见 配置管理网络参数。

图3-10 控制台管理 – 配置管理接口的网络参数
管理员配置RSAS的管理接口网络参数时，需要注意以下几点：
⚫ ipv4地址获取方式或ipv6地址获取方式配置为“dhcp”，按Enter键，RSAS将进行DHCP
自动获取。
⚫ 管理接口可以配置为IPv4或者IPv6，其中IPv4地址默认为192.168.1.1/24。
⚫ 如果管理员的主机与RSAS的管理接口配置在不同的网段，则还需要添加管理口路由。
⚫ 协商模式是必填项，必须为接口配置协商模式。
⚫ 配置完毕，选择“ok”即刻生效。

管理员配置RSAS的扫描接口网络参数时，需要注意以下几点：
⚫ ipv4地址获取方式或ipv6地址获取方式配置为“dhcp”，按Enter键，RSAS将进行DHCP
自动获取。
⚫ 扫描接口可以配置为IPv4或者IPv6，配置完毕，选择“ok”，配置立即生效。
⚫ 若需RSAS设备在线升级，网关和DNS服务器必须正确配置。
⚫ 网络参数的填写格式必须正确（例如：网络掩码255.255.255.0）。
添加管理口路由
如果需要人为的指定对某一个网络访问时所要经过的路径，可以通过手工添加或者删除静态
路由来实现。
在控制台管理的主菜单中，进入 网络配置 > 添加管理口路由，如图3-12所示。
图3-12 控制台管理 – 添加管理口路由
管理员添加RSAS的管理口路由时，需要注意以下几点：
⚫ 设备默认接口eth0为管理口，为其配置路由参数。
⚫ 设置IP地址、网络掩码以及网关后，选择“ok”，配置的路由即刻生效。
⚫ 如果单击ok出错，请确认添加的IP和网关等信息是否正确。
删除管理口路由
在控制台管理的主菜单中，进入 网络配置 > 删除管理口路由，如图3-13所示。

管理员删除RSAS的管理口路由时，需要注意以下几点：

添加扫描口路由
在控制台管理的主菜单中，进入 网络配置 > 添加扫描口路由，添加扫描口路由的方法与添
加管理口路由相同，请参考 添加管理口路由 的介绍。
删除扫描口路由
在控制台管理的主菜单中，进入 网络配置 > 删除扫描口路由，删除扫描口路由的方法与删
除管理口路由相同，请参考 删除管理口路由 的介绍。
