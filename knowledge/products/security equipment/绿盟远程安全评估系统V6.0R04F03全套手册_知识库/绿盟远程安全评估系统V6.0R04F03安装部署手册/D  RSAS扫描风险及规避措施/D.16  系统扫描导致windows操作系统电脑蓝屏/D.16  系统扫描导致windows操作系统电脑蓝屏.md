解决办法
进入 系统管理 > 配置 > 系统配置，端口屏蔽配置中添加打印相关端口，屏蔽掉不做扫描。
D.15 评估任务导致东方通中间件 80 端口服务挂掉
下发评估任务导致目标机80端口服务宕掉。
原因
插件“WWW服务器畸形HTTP请求方法413错误页面跨站脚本漏洞【原理扫描】”导致，该
漏洞的检测逻辑是在http头部设置一个很大的长度（Content-Length:
99999999999999999999999999999999999999），由于目标服务存在该漏洞或没有对特殊情况
做兼容，在Content-Length比实际的长度大情况下，由于实际提供的没有这么大，会无响应
直到超时，最终导致服务挂掉。
解决办法
联系东方通中间件厂商修复相关漏洞或者做兼容配置。
RSAS 端规避方案
扫描有东方通中间件目标机时，可自定义漏洞插件模板，剔除掉“WWW服务器畸形HTTP
请求方法413错误页面跨站脚本漏洞【原理扫描】”后执行扫描。
D.16 系统扫描导致 windows 操作系统电脑蓝屏
原因
目标机存在微软系统相关漏洞未更新微软补丁至最新导致
解决方案
保持目标机微软补丁为最新。
