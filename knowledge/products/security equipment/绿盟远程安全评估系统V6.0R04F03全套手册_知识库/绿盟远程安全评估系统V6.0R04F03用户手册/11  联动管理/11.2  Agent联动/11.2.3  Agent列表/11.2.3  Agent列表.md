11.2.3 Agent 列表
下载Agent时会绑定RSAS的IP地址，在目标机中完成Agent安装后，Agent会根据
RSAS的IP地址自动上报心跳至RSAS完成注册。注册完成后，Agent列表中会自动显
示相应的目标机信息。
查看 Agent 使用状态
进入 联动管理 > Agent联动 > Agent列表 页面，可以查看Agent的使用状态。
⚫ 授权数：RSAS可管理的Agent的数量，等于 查看授权注册信息 中的“全量Agent
扫描授权数”。
⚫ 注册数：所有已经注册的Agent数量。
⚫ 在线数：已注册中在线的Agent数量。
⚫ 三者关系：授权数≥注册数≥在线数。
管理可注册范围
配置可注册IP范围后，新目标机只有在可注册IP范围、安装Agent，且RSAS的“注
册数”小于“授权数”，才会显示在Agent列表中。对于在配置可注册IP范围前已经注
册的但不在配置的IP范围内的Agent，会在数分钟内变为离线状态。
进入 联动管理 > Agent联动 > Agent列表 页面，单击【可注册IP范围】按钮，配置
允许注册的目标机的IP范围即可。格式请参见界面说明，*表示不限制。
管理 Agent
进入 联动管理 > Agent联动 > Agent列表 页面，管理员可以对Agent进行查询、筛选
不同状态的Agent和如下操作：
⚫ 查看主机报表：单击图标 ，若RSAS已经对Agent相应的目标机执行过漏洞配
置任务，则可以查看最后一次扫描结果的主机报表。
⚫ 删除：删除列表中的Agent；删除后，释放授权数，但过段时间后：
− 在可注册IP范围内的主机，数分钟后会继续显示在Agent列表中。
− 不在可注册IP范围内的主机，删除后不再显示在 Agent列表中。

日志管理
只有审计管理员auditor拥有日志管理的权限。RSAS对管理员登录及在设备上的所有操
作进行严格审计，并以日志形式输出审计记录。RSAS输出的日志类型主要包括四类：
登录日志、操作日志、异常日志和升级日志。
本章主要内容包括：
功能 描述
日志审计 介绍审计员auditor查询、备份以及清空日志的具体方法。
日志配置 介绍审计员auditor配置日志阈值、日志备份方式的具体方法。
12.1 日志审计
查询日志
进入 日志管理 > 日志审计 页面，设定查询条件，查看符合查询条件的日志即可。通
过查看各类日志信息，可以帮助审计员获取以下RSAS相关信息：
⚫ 登录日志
了解RSAS设备的登录情况（包括登录成功、登录失败以及退出登录），及时发现
未授权用户的尝试登录。
⚫ 操作日志
审查所有管理员的操作行为，包括证书管理、网络配置、用户管理等等。
⚫ 异常日志
系统日志记录了解系统异常情况（包括网络不通、引擎异常、系统异常重启以及导
致系统无法正常工作的各种异常日志），有助于解决系统错误。
⚫ 升级日志
RSAS的系统版本及版本升级情况。
