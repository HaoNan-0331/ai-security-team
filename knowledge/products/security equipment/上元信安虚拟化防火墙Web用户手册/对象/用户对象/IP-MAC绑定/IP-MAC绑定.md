# IP-MAC绑定

IP-MAC绑定功能可对ARP欺骗攻击进行防御，启用该功能后，通过设备的MAC和IP需要和绑定关系保持一致，否则报文将被丢弃。IP-MAC绑定对二层数据和三层数据生效。

在系统菜单点击“对象>用户对象>IP-MAC绑定”，进入IP-MAC绑定配置页面。可以实现对IP-MAC绑定的新建、修改、删除、查询、重置、导入和导出功能。

IP-MAC绑定页面下点击<新建>，或在右侧“操作”列下点击图标修改已有的IP-MAC绑定。
