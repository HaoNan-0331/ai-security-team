# LDAP用户同步

轻型目录访问协议（Lightweight Directory Access Protocol，简称LDAP），虚拟化防火墙还支持LDAP用户同步，将LDAP服务器的用户同步到本地。

在系统菜单点击“对象>用户对象>LDAP用户同步”，进入LDAP用户同步配置页面，可以实现对LDAP用户同步配置的新建、修改、删除、同步和查看详细信息功能。

在LDAP用户同步页面下点击<新建>创建新的LDAP用户同步，或在右侧“操作”列下点击图标，修改已有的LDAP用户同步配置。
