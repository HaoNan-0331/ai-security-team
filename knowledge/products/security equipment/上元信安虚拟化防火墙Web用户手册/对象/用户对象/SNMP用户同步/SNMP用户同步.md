# SNMP用户同步

SNMP用户同步作用于跨三层IP-MAC绑定，可将三层交换机中ARP表同步到设备，对同步来的信息进行IP地址和MAC地址绑定。

在系统菜单点击“对象>用户对象>SNMP用户同步”，进入SNMP用户同步配置页面，可对SNMP用户同步执行新建、编辑、删除、同步和查看详细信息功能。

在SNMP用户同步页面下点击<新建>创建新的SNMP用户同步,或在右侧“操作”列下点击图标修改已有的SNMP用户同步任务。

SNMP用户同步的配置项与详细说明如下：

创建完成后，在SNMP用户同步页右侧“操作”列点击图标立即执行SNMP用户同步任务，点击图标显示同步结果。

点击右侧“操作”列按钮，查看同步结果。

进入点击同步结果页签，点击右侧“操作”列按钮，查看同步详情。

在同步详情页面，选择需要进行IP MAC绑定的条目，点击“操作”列右侧按钮，可执行绑定操作。

可跳转页面至“对象>用户对象>IP-MAC绑定”查看被绑定的内容。
