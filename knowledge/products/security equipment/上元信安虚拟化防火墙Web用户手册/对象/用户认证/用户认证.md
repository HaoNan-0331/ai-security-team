# 用户认证

认证是为保护系统的隐私数据与资源，使用户合法的访问内部系统的资源的一种方式。虚拟化防火墙支持本地认证、短信认证、远程认证、Portal认证、单点登录等多种认证方式，也支持多种认证方式组合使用，为用户带来高效与便捷。

启用本地认证功能后，用户打开浏览器访问HTTP页面会被定向到本地认证登录页面，输入正确的用户名和密码即可通过认证。

在系统菜单点击“对象>用户认证>本地认证”，进入本地认证配置页面。

流量超时：用户连续一段时间无流量产生，此段时间超过设置的超时时间，用户需重新认证。

在用户策略中启用 Portal 认证功能后，未认证用户上网时，设备强制用户登录到特定站点，用户可以免费访问其中的服务。当用户需要使用互联网中的其它信息时，必须在Portal网站进行认证，只有认证通过后才可以使用互联网资源。Portal业务可以提供方便的管理功能，Portal网站还可以开展广告、业务通知、个性化的业务等。
