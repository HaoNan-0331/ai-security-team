# 资产

该功能所描述的资产为信息资产，资产的信息包括资产IP、描述、用户、部门、重要度、操作系统、可用服务、来源、状态等，通过主动添加、被动流量发现和关联功能，实现资产统计。（端口扫描与EDR联动添加资产的方式请参考端口扫描与EDR策略这两个功能）。

在系统菜单点击“对象>资产>资产管理”，显示出设备上统计的所有资产，可对资产进行新建、编辑、删除、查询、导出和导入功能。

在资产管理页面点击<新建>创建一个新的资产，或者在右侧“操作”列下点击图标编辑已存在的资产信息。

资产管理配置项及详细说明如下：

点击<导出>，可将已存在的资产全部以表格的形式导出。

点击<导入>，选择导出的.csv文件，可将文件中的资产全部导入。

点击<查询>，可以根据资产IP、描述、用户、部门、重要度、操作系统、可用服务、状态字段来进行精准查询。

点击<清除>，可清除当前所有的资产。

该功能主要对流量发现方式统计资产的范围进行限制，用户可以通过配置排除范围与识别范围来精准的统计某地址范围内的资产。

在系统菜单点击“对象>资产>资产识别设定”，进入资产识别设定配置页面，开启“启用”后可配置识别地址、排除地址。
