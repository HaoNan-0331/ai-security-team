# 地址对象

在系统菜单点击“对象>地址>地址对象”，进入地址对象配置页面，可查看设备上已有的地址对象，其中前面8个地址对象是设备内置的IPv4地址对象，分别是any(任意地址)、ISP_CMCC.dat（中国移动）、ISP_UNICOM.dat(中国联通)、ISP_CT.dat(中国电信)、ISP_CTT.dat（中国铁通）、ISP_CERNET.dat(教育网)、ISP_INTL.dat（印度）和ISP_CHINA.dat(中国全域 )。用户无法修改或删除内置的IPv4地址对象。

在已创建的域名地址对象右侧“操作”列下点击图标查看域名对应的解析地址。

在地址对象页面下点击<新建>，创建IPv4的地址对象，或在右侧“操作”列下点击图标修改已有的地址对象。
