# 升级与重启

支持对设备版本进行升级、支持查看版本升级记录；同时支持特征库版本自动升级及手动升级；支持页面重启设备、恢复出厂设置。

固件升级中不仅支持设备版本升级，还支持双系统备份，且在系统切换中可实现配置的自动迁移。该场景是为保证设备的稳定性，在单个设备上进行双系统备份，以达到一个系统崩溃，备用系统可以切换运行的特性。

系统菜单点击“系统>系统维护>升级与重启>固件升级”，进入固件升级配置页签。

软件镜像用于升级设备版本，点击<选取文件>按钮来选择版本文件，点击<升级版本>开始版本升级操作，成功后需要手动重启设备，使新版本生效，可在下方升级历史中查看设备升级历史记录。

系统备份用于双系统备份，点击<备份当前系统>按钮进行系统备份，在启动选项子目录下，点击下次启动版本右侧下拉框，可以展示备份版本。

在系统菜单点击“系统>系统维护>升级与重启>特征库版本升级”，进入特征库版本升级页签。

选择需要升级的特征库，点击<选取文件>按钮选择特征库文件，选择后点击<升级>按钮开始升级操作。

自动升级需指定升级服务器，默认提供升级所用服务器，可根据需要进行修改，可进行定期升级或立刻升级。
