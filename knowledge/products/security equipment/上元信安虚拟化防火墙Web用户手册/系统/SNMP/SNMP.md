# SNMP

简单网络管理协议（Simple Network Management Protocol，简称SNMP），是标准IP网络管理协议。该协议使网络管理员可管理网络中设备，发现当前网络中存在的问题。

在系统菜单点击“系统>SNMP”进入SNMP配置页面，共分为2个页签，SNMP页签下用于配置SNMP基本信息，SNMP用户页签下用于配置SNMPv3版本所需要的用户及密码等信息。

在系统菜单点击“系统>SNMP>SNMP”进入SNMP配置页签。

当启用SNMPv3版本后，如需采用v3版本获取设备信息，设备必须配置SNMP用户，并且SNMP服务器软件中所设置的用户信息与设备中用户信息必须保持一致。

在系统菜单点击“系统>SNMP>SNMP用户”，进入SNMP用户配置页签。

在SNMP用户配置页签下，点击<新建>创建SNMP用户，也可以在右侧“操作”列点击删除SNMP用户信息。
