# 非法外联学习

当用户访问网络服务时，可以在非法外联策略配置动作允许的外联规则，通过非法外联学习用户访问的所有服务器地址。学习完成后，将期望拒绝访问的服务器主机添加到非法外联策略配置的服务器对象中，用户就无法访问通过非法外联学习获取的网络服务器主机。

系统菜单点击“策略>其他防护>非法外联防护>非法外联学习”，进入非法外联学习页签。

在非法外联学习页面选择需要学习的服务器对象、学习时长等配置后，点击<应用>下发配置，点击开始学习中的<开始>按钮，开始学习，点击<停止>按钮，则停止学习。
