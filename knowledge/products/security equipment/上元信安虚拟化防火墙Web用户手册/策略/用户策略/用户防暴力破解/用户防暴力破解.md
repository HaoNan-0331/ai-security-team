# 用户防暴力破解

虚拟化防火墙支持用户防暴力破解功能，当监测到SSL VPN、IPSec-VPN、Portal认证、远程认证、本地认证等认证行为时，对认证失败次数进行监控，当一分钟内连续失败次数达到阈值后，会进行认证IP的锁定，从而避免非法人员利用工具破解用户密码，给用户资产带来不可预计的损失。

在系统菜单点击“策略>用户策略>用户防暴力破解>用户防暴力破解”，进入用户防暴力破解配置页签，分别“启用”同IP用户登录和同名用户登录，看到如下配置。

用户防暴力破解配置项及详细说明如下：

完成用户防暴力破解配置后，在系统菜单点击“策略>用户策略>用户防暴力破解>用户锁定”，切换到用户锁定页签，查看锁定的IP地址，点击操作列的解绑按钮，对锁定的用户进行解绑。

提示：

同名用户登录统计认证失败次数的顺序在同IP用户登录之前。
