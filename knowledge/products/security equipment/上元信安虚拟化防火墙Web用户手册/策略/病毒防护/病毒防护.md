# 病毒防护

虚拟化防火墙支持病毒检测防护，支持对常用协议类型（HTTP、FTP、IMAP、SMTP、POP3）进行病毒扫描及防护，也可指定文件类型和自定义文件类型对病毒进行扫描和防护。

在系统菜单点击“对象>防护配置模板>病毒防护”，点击可查看设备支持的病毒数量。可以定期升级病毒特征库，升级方式请参考“升级与重启”。

病毒防护功能作为策略模板，需要被一体化策略引用才能生效，配置请参考“防火墙策略”。

在系统菜单点击“策略>病毒防护>病毒防护模板”，进入病毒防护模板配置页面，点击<新建>创建新的病毒防护策略。

返回病毒防护模板页面，右侧“操作”列点击编辑已有病毒防护策略的描述、事件集、日志内容。也可以点击复制病毒防护策略。

默认扫描文件不开启，我们能够对启用的文件类型的病毒文件进行扫描，提高效率。开启扫描所有文件，除文件类型支持之外的病毒类型，也可进行扫描，开启tar文件扫描可能会影响性能。

系统菜单点击“策略>病毒防护>扫描文件设定>扫描文件设定”，进入扫描文件设定配置。
