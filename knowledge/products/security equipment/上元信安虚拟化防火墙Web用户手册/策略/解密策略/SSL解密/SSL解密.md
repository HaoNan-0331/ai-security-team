# SSL解密

随着市场网络安全的发展，更多的客户将原有的明文业务逐步迁移到SSL协议，能够很大程度保证客户访问网络的安全，数据不会轻易被第三方窥视。网络管理员也很难得到内网用户访问外网的相关数据，对企业内网有很大的风险。基于此问题，防火墙能够对SSL的流量进行解密，并完成明文流量的审计。

在系统菜单点击“策略>解密策略>SSL解密>SSL解密”，进入SSL解密配置页签。
