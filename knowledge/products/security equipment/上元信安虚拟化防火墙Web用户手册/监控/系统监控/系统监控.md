# 系统监控

系统监控功能可监控设备当前在线用户、接口信息、设备健康、健康检查等信息。

在系统菜单点击“监控>系统监控>在线用户统计”，进入在线用户统计页面，查看设备上所有用户。

在在线用户统计页面右侧“操作”栏点击查看该用户的流量统计。

在在线用户统计页签,点击右侧“操作”栏可以冻结该用户，被冻结后的用户在冻结时间内无法通过防火墙上网。

在系统菜单点击“监控>系统监控>接口信息统计”，进入接口信息统计页面，查看设备所有接口的信息统计内容。

设备健康统计用于展示一定时间段内设备系统资源占用情况，如：通过设备的上下行流量大小、设备连接数、CPU利用率及内存利用率，用户可根据需要查看不同时间段内设备健康情况。

在系统菜单点击“监控>系统监控>设备健康统计”，进入设备健康统计配置页面，可直观查看到所有设备健康统计信息，默认所有信息统计时间为最近半小时。

设备健康统计的配置项与详细说明如下：

设备流量：统计选定时间内通过设备的上下行流量，可通过点击<最近半小时>下拉菜单查看可选时间段，切换时间段可变更查询范围，可通过点击<上行><下行>按钮，进行单独展示上行或下行流量。

连接数统计：统计选定时间内设备中存在的连接数，包括总连接及新建连接数，可通过点击<最近半小时>下拉菜单查看可选时间段，切换时间段可变更查询范围，可通过点击<总连接><新建连接>按钮，单独展示总连接或新建连接。

CPU利用率：统计选定时间内设备CPU占用情况，可通过点击<最近半小时>下拉菜单查看可选时间段，切换时间段可变更查询范围。

内存利用率：统计选定时间内设备内存占用情况，可通过点击<最近半小时>下拉菜单查看可选时间段，切换时间段可变更查询范围。

健康检查统计可对具体健康检查对象进行监控，统计被监控对象的链路丢包率、链路延时及延时抖动，以图表形式进行展示，可使用户直观的看到所选健康检查对象的链路状态。

在系统菜单点击“监控>系统监控>健康检查统计”，进入健康检查统计配置页面，可根据需求选择被引用的健康检查对象，根据出接口、统计时间，进行相应的图表展示。

健康检查统计的配置项与详细说明如下：

链路丢包率：以百分比为单位展示所选健康检查对象，在所选时间段内的具体丢包情统计到的百分比及具体统计时间，绘制图表进行展示。

链路延时：以ms为单位展示所选健康检查对象，在所选时间端内通信延时情况，并根据延时情况及时间段，绘制图表进行展示。

延时抖动：以ms为单位展示所选健康检查对象，在所选时间端内通信延时抖动情况，并根据延时抖动及抖动发生的时间段，绘制图表进行展示。
