# 静态路由

静态路由是用户在设备的路由表中手动添加的固定路由，适用于网络规模较小、拓扑结构固定的网络环境中。静态路由的优点是简单、高效、可靠、几乎不增加CPU负载。在所有路由中，静态路由的优先级最高，静态路由与动态路由发生冲突时，系统会以静态路由为准。

在系统菜单中点击“网络配置>IPv4路由>静态路由”，进入静态路由页面。

用户可在静态路由页面下，点击<新建>，创建新的静态路由，或在右侧“操作”列下点击图标修改已有的静态路由配置。

用户可在静态路由页面下，点击<全局配置>，存在等价路由情况下，可通过配置多路径选择对流量进行转发设置，支持根据源IP和根据连接两种选择方式。
