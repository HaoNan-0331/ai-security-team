# SSL VPN配置示例

组网需求

用户PC1通过SSLVPN Client拨号后与DUT1设备建立SSLVPN连接，VPN建立成功后PC1可以成功访问DUT1后的内网PC2。

DUT1 ge0/0地址：5.5.5.1/24，ge0/2地址：10.1.1.1/24。

Client IP地址：4.4.4.4/24。

PC2地址：10.1.1.2/24,默认网关为10.1.1.1。

配置步骤

DUT1：进入“网络>接口配置>物理接口”配置接口，ge0/0的IP地址配置5.5.5.1/24。

ge0/2的IP地址配置为10.1.1.1/24。

进入“对象>用户对象>用户”，点击新建，名称：test密码为123456。

DUT配置SSL VPN，进入“网络>SSL VPN>SSL VPN配置”，打开启用和多点登录开关，输入隧道地址11.1.1.1/24,地址池11.1.1.0/24，DNS1输入114.114.114.114，DNS2输入8.8.8.8，子网路由输入10.1.1.0/24。

PC配置

在PC上安装客户端。

打开客户端，填写服务器地址10.1.1.1及服务器端口号8443，输入用户名：test，用户密码：123456，点击登录。

登录成功,在监控页面可以查看到test用户登录信息。

通过client端PC1 ping PC2端IP地址，可以正常通信。

注意：

SSLVPN功能要能正常使用需要在VPN管理中开启VPN用户中心后方可使用。
