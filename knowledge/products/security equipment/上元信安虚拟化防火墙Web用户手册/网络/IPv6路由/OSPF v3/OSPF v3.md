# OSPF v3

OSPFv3是OSPF版本3的简称，是在OSPFv2基础上开发的用于IPv6网络的路由协议。随着IPv6网络的建设，同样需要动态路由协议为IPv6报文的转发提供准确有效的路由信息。基于此，IETF在保留OSPFv2优点的基础上针对IPv6网络修改形成OSPFv3。OSPFv3主要用于在IPv6网络中提供路由功能，是IPv6网络中路由技术的主流协议。

在系统菜单点击“网络>IPv6路由>OSPF v3>OSPF v3”，进入OSPF v3页签，显示OSPF v3功能配置信息。
