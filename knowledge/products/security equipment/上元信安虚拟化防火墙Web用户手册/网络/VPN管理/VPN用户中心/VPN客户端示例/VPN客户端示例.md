# VPN客户端示例

从SSL VPN门户页面下载SSL VPN客户端后，参考以下步骤安装、运行客户端：

双击安装包并按照提示安装客户端程序，具体信息请参考下图：

安装完成后，点击运行客户端程序，具体信息请参考下图：

输入服务器地址、服务器端口、用户名、用户密码后，点击<登录>，进行连接。

成功创建连接后，任务栏将会显示以下图标，具体信息请参考下图：

右键图标会显示客户端设置页面，点击<关于版本>即可看到版本号，具体信息请参考下图：

在防火墙的系统菜单中点击“网络>VPN管理>VPN用户中心>VPN客户端配置”中选择<导入本地>上传V11.11的压缩包（Windows 64位为例），具体信息请参考下图：

客户端断开连接，重新登录，会进行版本号校验，具体信息请参考下图：

点击<下载>按钮，会自动跳转网页进行下载，具体信息请参考下图：

下载成功后即可直接覆盖安装进行连接。

提示：

卸载SSL VPN客户端时建议使用系统控制面板卸载程序进行卸载。
