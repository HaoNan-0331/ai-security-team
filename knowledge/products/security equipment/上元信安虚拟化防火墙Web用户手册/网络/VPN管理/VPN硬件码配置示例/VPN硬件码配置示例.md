# VPN硬件码配置示例

组网需求

如下图所示，设备的ge0/1和ge0/2接口以路由方式部署在网络中。用户需要设备提供SSL VPN服务，并限制特定的互联网用户通过SSL VPN访问内网的Web和FTP服务器。

配置步骤

在系统菜单点击“网络>SSL VPN>SSL VPN配置”中配置SSL VPN的配置，具体信息请参考下图：

在系统菜单点击“对象>用户对象>用户”，进入用户页面，点击<新建>创建用户，具体信息请参考下图：

在系统菜单点击“网络>VPN管理>VPN用户中心”，进入VPN用户中心页面，打开<用户中心>，端口默认为8443，具体信息请参考下图：

在系统菜单点击“网络>VPN管理>VPN用户认证”，进入VPN用户认证页面，打开<硬件特征码>，默认接入终端数量为3，具体信息请参考下图：

使用客户端输入服务器地址、服务器端口、用户名、用户密码后，点击<登录>，客户端连接成功，具体信息请参考下图：

在系统菜单点击“网络>VPN管理>VPN用户接入”，进入VPN用户接入页面查看，具体信息请参考下图：

设备记录客户端的终端类型和硬件特征码，点击后，即可剔除客户端，客户端会提示下线，具体信息请参考下图：
