# VPN管理

为提升用户在VPN远程办公使用过程中的安全性，在现有用户认证（用户名密码）的基础上，增加动态码认证因素，以此提升VPN用户接入认证安全，解决弱身份鉴别可能引发的内网信息泄漏隐患。

Virtual Private Network（虚拟专用网络，简称VPN）用户中心专为用户提供密码修改、客户端下载和VPN帮助文档查看功能。

在系统菜单点击“网络>VPN管理>VPN用户密钥”，进入VPN用户密钥管理页面。此页面支持显示本地用户和LDAP用户，具体创建用户请参考用户和LDAP用户同步。

每个用户默认无密钥，可以点击图标生成密钥，生成密钥后可以点击图标查看二维码，可以使用手机应用（FressOTP或者Authenticator）通过密钥或者二维码的方式获取动态口令。

点击<生成密钥>可以根据选择的多用户生成密钥。

点击<刷新>可以实时刷新用户密钥状态。

点击<查询>可以支持用户和密钥状态来进行查询。

查询内容配置项及详细说明如下：

点击<导出>可以根据文件类型选择（支持二维码和密钥）进行当前页和所有页的导出。

系统菜单点击“网络>VPN管理>VPN用户接入”，进入VPN用户接入管理页面，显示所有已经连接过的客户端类型和硬件特征码。

点击图标可以单个解绑此用户终端，如果在线则剔除下线。

点击<解绑>按钮可以进行多用户多终端的解绑，如果在线则剔除下线。

点击<刷新>可以实时刷新用户接入状态。

点击<查询>按钮，可以根据用户名称和终端类型进行组合查询。

组网需求

如下图所示，设备的ge0/1和ge0/2接口以路由方式部署在网络中。用户需要设备提供SSL VPN服务，并限制特定的互联网用户通过SSL VPN访问内网的Web和FTP服务器。

配置步骤

在系统菜单点击“网络>SSL VPN>SSL VPN配置”，进入SSL VPN配置页面，配置SSL VPN的配置，具体信息请参考下图：

在系统菜单点击“对象>用户对象>用户”，进入用户页面，点击<新建>创建用户，具体信息请参考下图：

在系统菜单点击“网络>VPN管理>VPN用户中心”，进入VPN用户中心页签，打开<用户中心>，端口默认为8443，具体信息请参考下图：

在系统菜单点击“网络>VPN管理>VPN用户认证”，进入VPN用户认证页面，打开<动态口令>，具体信息请参考下图：

在系统菜单点击“网络>VPN管理>VPN用户密钥”，进入VPN用户密钥页面，创建用户的密钥，点击图标来生成密钥。

使用手机应用（FreeOTP或者Authenticator）输入密钥或者扫描二维码获取动态口令。

获取动态口令后，使用客户端进行<登录>操作，就会弹出“请输入动态口令”的输入框，具体信息请参考下图：

输入动态口令后，点击<连接>，显示连接成功则登录成功，具体信息请参考下图：

提示：

开启动态口令，设备需要进行时间同步。
