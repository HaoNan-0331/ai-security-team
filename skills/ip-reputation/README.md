# IP信誉情报查询Skill

这是一个用于查询IP地址信誉情报的Claude Code skill，通过多个威胁情报源进行综合分析和风险评分。

## 功能特性

- **多情报源查询**：支持VirusTotal、AbuseIPDB、AlienVault OTX、Shodan、URLhaus等
- **权重打分系统**：根据情报源的可靠性和重要性配置权重
- **综合风险评估**：整合多个来源数据，计算0-100的风险评分
- **批量IP查询**：支持批量查询多个IP地址
- **威胁情报报告**：生成详细的威胁情报报告，包括威胁类型、安全建议等

## 安装

1. 将 `ip-reputation.skill` 文件复制到Claude Code的skills目录
2. 在Claude Code中skill会被自动加载

## 使用方法

### 基本查询

```
检查IP 8.8.8.8 的信誉
```

### 批量查询

```
批量检查以下IP地址的安全性: 8.8.8.8, 1.1.1.1, 45.142.212.14
```

### 安全事件响应

```
我们在安全事件中发现可疑IP 185.220.101.1，请调查这个IP的威胁情报
```

### 风险评估

```
评估IP 192.0.2.1 的风险等级
```

## 评分标准

| 评分范围 | 风险等级 | 说明 |
|---------|---------|------|
| 0-20 | 低风险 | 可信IP，无恶意活动记录 |
| 21-40 | 较低风险 | 基本可信，少量非关键报告 |
| 41-60 | 中等风险 | 存在可疑活动，需要关注 |
| 61-80 | 高风险 | 多个情报源报告恶意活动 |
| 81-100 | 严重风险 | 确认为恶意IP |

## 情报源配置

默认配置的情报源：

| 情报源 | 权重 | 可信度 |
|-------|------|--------|
| VirusTotal | 0.30 | 0.95 |
| AbuseIPDB | 0.25 | 0.90 |
| AlienVault OTX | 0.20 | 0.85 |
| Shodan | 0.15 | 0.80 |
| URLhaus | 0.10 | 0.75 |

可以在 `assets/config/sources.yaml` 中修改权重和启用/禁用情报源。

## 工作原理

1. 获取用户提供的IP地址
2. 验证IP格式（IPv4/IPv6）
3. 加载情报源配置
4. 对每个启用的情报源执行网页查询
5. 解析每个情报源返回的数据
6. 根据配置的权重和可信度计算综合评分
7. 生成详细的威胁情报报告

## 注意事项

- 某些情报源可能需要登录或有访问限制
- 网页结构变化可能导致解析失败
- 建议将查询结果缓存，避免重复查询
- 尊重各情报源的使用条款和速率限制
- 此skill仅用于授权的安全测试，不得用于非法用途

## 测试

使用以下IP测试skill的功能：

- `8.8.8.8` - Google DNS（可信IP）
- `1.1.1.1` - Cloudflare DNS（可信IP）
- `185.220.101.1` - Tor节点（中等风险）
- `45.142.212.14` - 已知恶意IP（高风险）

## 开发

如果您需要添加新的情报源：

1. 编辑 `assets/config/sources.yaml`，添加新的情报源配置
2. 更新 `references/sources.md`，添加情报源的解析说明
3. 根据需要调整 `assets/config/scoring.yaml` 中的评分规则
