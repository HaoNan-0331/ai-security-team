# 病毒防护 (AV)

## 概述

病毒防护模块提供病毒防护模板和扫描文件类型的配置管理功能。

## API 端点

| 功能 | 方法 | 端点 |
|------|------|------|
| **病毒防护模板** | 获取病毒防护模板配置 | GET | `/api/av-rule` |
| **病毒防护模板** | 新建病毒防护模板 | POST | `/api/av-rule` |
| **病毒防护模板** | 修改病毒防护模板 | PUT | `/api/av-rule` |
| **病毒防护模板** | 删除病毒防护模板 | DELETE | `/api/av-rule` |
| **文件类型** | 获取文件类型配置 | GET | `/api/av-filetype` |
| **文件类型** | 新建文件类型配置 | POST | `/api/av-filetype` |
| **文件类型** | 修改文件类型配置 | PUT | `/api/av-filetype` |
| **文件类型** | 删除文件类型配置 | DELETE | `/api/av-filetype` |

# 病毒防护模板

## 获取病毒防护模板配置

**端点**: `GET /api/av-rule`

**响应参数**:

| 参数名 | 类型 | 说明 |
|--------|------|------|
| name | String | 名称（1-63个字符） |
| enable | Number | 启用（0：关闭，1：启用） |
| action | Number | 动作（0：通过，1：阻断） |
| http | Number | 协议（0：关闭，1：开启） |
| ftp | Number | 协议（0：关闭，1：开启） |
| smtp | Number | 协议（0：关闭，1：开启） |
| imap | Number | 协议（0：关闭，1：开启） |
| pop3 | Number | 协议（0：关闭，1：开启） |
| apt_enable | Number | 沙箱检测（0：关闭，1：启用） |
| desc | String | 描述（0-127个字符） |
| refcnt | Number | 引用计数 |

**响应示例**:

```json
{
    "data": [
        {
            "name": "病毒防护模板1",
            "desc": "",
            "refcnt": "0",
            "http": "0",
            "smtp": "0",
            "imap": "0",
            "ftp": "0",
            "pop3": "1",
            "enable": "1",
            "action": "0",
            "apt_enable": "1"
        }
    ],
    "total": 1
}
```

## 新建病毒防护模板

**端点**: `POST /api/av-rule`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | String | 否 | 名称（1-63个字符） |
| enable | Number | 否 | 启用（0：关闭，1：启用） |
| action | Number | 否 | 动作（0：通过，1：阻断） |
| http | Number | 否 | 协议（0：关闭，1：开启） |
| ftp | Number | 否 | 协议（0：关闭，1：开启） |
| smtp | Number | 否 | 协议（0：关闭，1：开启） |
| imap | Number | 否 | 协议（0：关闭，1：开启） |
| pop3 | Number | 否 | 协议（0：关闭，1：开启） |
| apt_enable | Number | 否 | 沙箱检测（0：关闭，1：启用） |
| desc | String | 是 | 描述（0-127个字符） |

**请求示例**:

```json
{
    "name": "病毒防护模板1",
    "enable": "0",
    "action": "0",
    "http": "0",
    "ftp": "1",
    "smtp": "1",
    "imap": "1",
    "pop3": "0",
    "apt_enable": "1",
    "desc": "111"
}
```

**响应示例**:

下发正确返回空，下发错误返回code和str：

```json
{
    "code": "-1",
    "str": "Av策略病毒防护模板1已经存在"
}
```

# 扫描文件设定

## 获取文件类型配置

**端点**: `GET /api/av-filetype`

**响应参数**:

| 参数名 | 类型 | 说明 |
|--------|------|------|
| pattern | String | 文件类型（1-14个字符） |
| enable | Number | 是否使能（0：禁用，1：启用） |
| is_static | Number | 是否可删除（0：可以删除，1：不可以删除，默认为1） |

**响应示例**:

```json
{
    "data": [
        {
            "pattern": "*.exe",
            "enable": "1",
            "is_static": "1"
        },
        {
            "pattern": "*.bat",
            "enable": "1",
            "is_static": "1"
        }
    ],
    "total": 2
}
```

---

## 新建文件类型配置

**端点**: `POST /api/av-filetype`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pattern | String | 否 | 文件类型，必须以*.开头（1-14个字符） |

**请求示例**:

```json
{
    "pattern": "*.b"
}
```

---

## 修改文件类型配置

**端点**: `PUT /api/av-filetype`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pattern | String | 否 | 文件类型，必须以*.开头（1-14个字符） |
| enable | Number | 否 | 是否使能（0：禁用，1：启用） |

**请求示例**:

```json
{
    "pattern": "*.b",
    "enable": "0"
}
```

---

## 删除文件类型配置

**端点**: `DELETE /api/av-filetype`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pattern | String | 否 | 文件类型，必须以*.开头（1-14个字符） |

---

## 参数说明

### action 参数说明

| 值 | 说明 |
|----|------|
| 0 | 通过 |
| 1 | 阻断 |

### enable 参数说明

| 值 | 说明 |
|----|------|
| 0 | 禁用/关闭 |
| 1 | 启用/开启 |

### is_static 参数说明

| 值 | 说明 |
|----|------|
| 0 | 可以删除 |
| 1 | 不可以删除（默认） |
