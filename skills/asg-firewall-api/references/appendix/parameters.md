# 通用参数说明

## 概述

ASG防火墙API中有许多通用参数在多个接口中使用。本文档详细说明这些参数的含义和取值范围。

## URL 参数

### 通用URL参数

| 参数名 | 类型 | 必填 | 说明 | 可选值/范围 |
|--------|------|------|------|-------------|
| api_key | String | 是 | 认证Token | 从Web界面获取 |
| lang | String | 否 | 语言 | cn（中文）、en（英文），默认cn |
| page | Number | 否 | 当前页数 | ≥1，默认1 |
| pageSize | Number | 否 | 每页条数 | ≥1，默认10或更大 |

### 使用示例

```bash
# 基本请求
http://172.17.108.86/api/policy?api_key=YOUR_TOKEN

# 带语言参数
http://172.17.108.86/api/policy?api_key=YOUR_TOKEN&lang=en

# 分页查询
http://172.17.108.86/api/policy?api_key=YOUR_TOKEN&page=2&pageSize=20
```

## 协议类型参数

### protocol

**说明**: 指定IP协议版本

| 值 | 说明 | 适用场景 |
|----|------|----------|
| 0 | IPv6 | IPv6网络环境 |
| 1 | IPv4 | IPv4网络环境（默认） |
| 2 | IPv4 | 部分接口使用 |

**使用示例**:

```json
{
    "protocol": "1"
}
```

## 启用/禁用参数

### enable

**说明**: 控制功能或对象的启用状态

| 值 | 说明 |
|----|------|
| 0 | 禁用/关闭/不启用 |
| 1 | 启用/开启 |

**使用场景**:
- 策略启用状态
- 接口启用状态
- 功能开关
- 日志开关

**使用示例**:

```json
{
    "enable": "1"
}
```

## 策略动作参数

### mode / action

**说明**: 防火墙策略的动作类型

| mode值 | action值 | 说明 |
|--------|----------|------|
| "1" | - | PERMIT（允许） |
| "2" | - | DENY（拒绝） |
| - | "0" | 通过（病毒防护） |
| - | "1" | 阻断（病毒防护） |

**使用示例**:

```json
{
    "mode": "1"
}
```

## 日志相关参数

### syslog

**说明**: 控制是否记录日志

| 值 | 说明 |
|----|------|
| 0 | 关闭日志 |
| 1 | 开启日志 |

### log_level

**说明**: 日志级别，决定日志的严重程度

| 值 | 说明 | 英文 |
|----|------|------|
| 0 | 紧急 | Emergency |
| 1 | 告警 | Alert |
| 2 | 严重 | Critical |
| 3 | 错误 | Error |
| 4 | 警示 | Warning |
| 5 | 通知 | Notice |
| 6 | 信息 | Informational |

**使用示例**:

```json
{
    "syslog": "1",
    "log_level": "6"
}
```

## 策略ID参数

### id

**说明**: 策略或规则的唯一标识

| 参数 | 说明 |
|------|------|
| 取值范围 | 1-10000 |
| 特殊值 | 0（自动分配ID） |

**使用示例**:

```json
{
    "id": "1"
}
```

**注意事项**:
- 添加策略时，id填0表示自动分配
- 修改/删除策略时，id必须指定已存在的策略ID
- 同一类型策略的ID必须唯一

## 地址类型参数

### addr_type / type

**说明**: 地址对象的类型

| 值 | 说明 |
|----|------|
| 0 | IPv4地址 |
| 1 | IPv6地址（部分接口） |

### item[].type

**说明**: 地址项的类型

| 值 | 说明 | 示例 |
|----|------|------|
| 0 | 主机 | 192.168.1.1 |
| 1 | 子网 | 192.168.1.0/24 |
| 2 | 范围 | 192.168.1.1-192.168.1.100 |

**使用示例**:

```json
{
    "name": "test_addr",
    "type": 0,
    "item": [
        {"host": "192.168.1.1", "type": 0},
        {"net": "192.168.1.0/24", "type": 1},
        {"range1": "192.168.1.1", "range2": "192.168.1.100", "type": 2}
    ]
}
```

## 接口参数

### if_in / if_out

**说明**: 流量入接口和出接口

| 参数 | 说明 | 常用值 |
|------|------|--------|
| if_in | 入接口 | ge0/0, ge0/1, any |
| if_out | 出接口 | ge0/0, ge0/1, any |

**使用示例**:

```json
{
    "if_in": "ge0/0",
    "if_out": "any"
}
```

## 移动操作参数

### move / mv_opt

**说明**: 策略移动操作

| 值 | 说明 |
|----|------|
| 1 | 移动到指定策略ID之前 |
| 2 | 移动到指定策略ID之后 |

**配合参数**:
- dst_id / refer_id: 目标策略ID

**使用示例**:

```json
{
    "id": "2",
    "move": "1",
    "dst_id": "1"
}
```

## 黑名单特定参数

### ip_type

**说明**: 黑名单IP地址类型

| 值 | 说明 |
|----|------|
| 1 | IPv4地址 |
| 2 | IPv6地址 |

### age

**说明**: 黑名单有效期

| 值 | 说明 |
|----|------|

    "enable": "1"

| 值 | 说明 |

| cur_day | 当天 |
| cur_week | 本周 |
